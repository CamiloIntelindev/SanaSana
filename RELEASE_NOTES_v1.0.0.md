## Highlights

## Installation
1. Copy `sanasana/` into `wp-content/plugins/`.
2. Activate the plugin in WordPress Admin.
3. Verify assets load correctly.

## Notes
- Secrets are not tracked: `wp-config.php` and `.env*` are ignored.
- Main branch: `main`. Tag: `v1.0.0`.
